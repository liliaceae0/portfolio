Homelab : laboratoire d'expérimentation personnel

Montage d'un serveur physique avec Proxmox + VMware, pfSense, Windows Server, Linux et Docker pour expérimenter librement les technologies virtualisation, réseau et sécurité.

Contexte

Projet personnel avec un camarade pour monter un laboratoire à la maison, permettant d'expérimenter librement des technologies sans risque pour une infrastructure de production.

Problématique

Comment créer un environnement sandbox à domicile pour expérimenter la virtualisation, le réseau et la sécurité en dehors des cours ?

Environnement technique

Hyperviseurs
  • Serveur (PC reconditionné)
  • Proxmox + VMware
VMs
  • pfSense (firewall)
  • Windows Server
  • Linux, Docker

Étapes réalisées

  1. 1Installation du serveur physique et configuration hardware.
  2. 2Déploiement de Proxmox comme hyperviseur principal.
  3. 3Création de VMs : AD (Windows Server), pfSense (firewall), serveur web (Linux).
  4. 4Configuration du réseau interne (VLANs, routage).
  5. 5Expérimentation de Docker pour le déploiement de conteneurs.

Bilan

Sandbox d'expérimentation fonctionnel permettant de tester toutes les technologies vues en cours. Accélération de l'apprentissage et compréhension approfondie des architectures réseau.
